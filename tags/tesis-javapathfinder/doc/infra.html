Bugs

SourceForge is our main development site, so please use the bug tracking system on SourceForge.

When submitting new bugs, please make sure to include the following information:

  • java version and platform
  • any modified configuration files (jpf.properties, default.properties etc. - attach file)
  • how you invoke JPF (command line or Eclipse args)
  • stack dumps (if any, please set 'jpf.print_exception_stack=True')
  • failing application (attach at least a classfile, if it's yours)
  • if it's a failing regression test - attach the corresponding Test*.txt from the 'build/test' dir

Thank's for making our life easier that way!
